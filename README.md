# paheisub
Pahe.In dan Isubtitles downloader

# how to install
```
apt install python
apt install git
git clone https://github.com/KANG-NEWBIE/paheisub-dl
cd paheisub-dl
python -m pip install requests click tqdm bs4
```
<b>Usage:</b> ```python paheisub.py```
